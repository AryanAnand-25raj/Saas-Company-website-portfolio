# FlowPilot SaaS Website

A responsive, modern landing page for a fictional SaaS product company. It is built with plain HTML, CSS, and JavaScript, so there are no packages to install.

## Run the project

Open `index.html` directly in a browser, or start a local server:

```powershell
cd saas-company-website
python -m http.server 5500
```

Then open `http://localhost:5500`.

## Project files

- `index.html` contains the page structure and content.
- `styles.css` contains the complete responsive design.
- `script.js` controls mobile navigation, FAQ accordions, pricing selection, and scroll animations.

## Customize

Search for `FlowPilot` in `index.html` to change the company name. Brand colors are stored as CSS variables at the top of `styles.css`.
